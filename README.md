# Image Caption Generator

This project generates natural language captions for uploaded images using the BLIP vision-language model and Streamlit.

## Features

- Upload an image through a web interface
- Generate an automatic caption
- Clean and simple Streamlit UI
- Uses a pretrained BLIP image captioning model

## Technologies Used

- Python
- Streamlit
- Hugging Face Transformers
- PyTorch
- Pillow

## How It Works

The application uses the `Salesforce/blip-image-captioning-base` model to process an image and generate a textual description.

## Installation

```bash
pip install -r requirements.txt
```
